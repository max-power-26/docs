## Performing Web Searches and Checking Weather via Voice

By talking to Veta, you can search for information on the internet or immediately check the weather and the current time. This feature can be used **without connecting a Google account**.

## Web Search

If you ask a question you're curious about, Veta will find the latest information on the internet and summarize it for you.

**Speaking Examples**

<CardGroup cols={2}>

<Card title="General Search" icon="magnifying-glass">
**"Search for Apple's stock price."**

**"Tell me the latest news on Samsung Electronics."**
</Card>

<Card title="Work-related Search" icon="briefcase">
**"Search for pharmaceutical distribution regulations."**

**"Tell me the GMP certification standards."**
</Card>

</CardGroup>

Veta will summarize the search results for you. If you need more detailed information, simply follow up by saying, "Tell me more details."

## Checking the Weather

You can check the weather for any location instantly.

**Speaking Examples**

<CardGroup cols={2}>

<Card title="Current Weather" icon="sun">
**"How is the weather in Seoul?"**

**"Tell me today's weather."**
</Card>

<Card title="Weather in Other Areas" icon="cloud">
**"How is the weather in Busan?"**

**"Tell me tomorrow's weather in Jeju Island."**
</Card>

</CardGroup>

Veta will inform you of the current temperature, weather conditions, and the feels-like temperature.

## Checking the Current Time

You can check the current time using your voice.

**Speaking Examples**

<CardGroup cols={1}>

<Card title="Check Time" icon="clock">
**"What time is it right now?"**
</Card>

</CardGroup>

## Usage Examples

Web search is particularly useful when you want to quickly look up information while working.

<CardGroup cols={2}>

<Card title="Check Client Info" icon="building">
You can quickly search for information about a client before a meeting.

**"Search for company info on Taejun Pharmaceuticals."**
</Card>

<Card title="Check Weather Before a Trip" icon="umbrella">
You can check the weather and pack an umbrella before heading out for a business trip.

**"Is it raining today?"**
</Card>

</CardGroup>

<Tip>
If the web search returns too many results, try making your question more specific. Instead of just saying "Pharmaceuticals," asking something like "Pharmaceutical wholesale business license standards 2026" will yield much more accurate results.
</Tip>

<Tip>
Checking the weather and time can be done instantly without a Google account connection. Try it right after installing the app!
</Tip>
