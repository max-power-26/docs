## You Can Also Use Veta in Telegram

If you are already using Telegram, you can use the Veta AI bot directly within Telegram without having to switch messengers.

Telegram is a widely used free messenger app worldwide. You can search for and install "Telegram" from the App Store or Play Store.

## Connecting to Telegram

Here are the 5 steps to connect your ClawOnPhone account with Telegram.

<Steps>
  <Step title="Search for the Bot in Telegram">
    Open the Telegram app and enter **@clawOnPhoneBot** into the top search bar. Please select the ClawOnPhone Bot from the search results.
  </Step>
  <Step title="Enter the /start Command">
    Once you enter the bot chat room, you will see a **Start** button at the bottom. Press the button or manually type directly **/start**. The bot will send you a welcome message along with pairing instructions.
  </Step>
  <Step title="Generate a Pairing Code in ClawOnPhone">
    Open the ClawOnPhone PWA app and press the **TG button** on the screen. A 6-character pairing code will be generated. This code is valid for 5 minutes.
  </Step>
  <Step title="Enter the Pairing Code in Telegram">
    Return to the ClawOnPhone Bot chat room in Telegram and enter **/pair 6-character-code**. For example, if the code is A3B7K9, you would type **/pair A3B7K9**.
  </Step>
  <Step title="Connection Complete">
    If you see the message "Connection complete," you were successful. Now you can directly send messages to Veta in Telegram.
  </Step>
</Steps>

## How to Use

Once the connection is complete, simply send **messages in natural language** in the ClawOnPhone Bot chat room on Telegram.

You can use it exactly as you would when talking to Veta in ClawTalk. Type normally as you would speak, like "Tell me today's schedule" or "Check unread emails."

Wait a moment while Veta processes your request, and it will reply with the results in text.

## Main Commands

<CardGroup cols={2}>
  <Card title="/start" icon="play">
    Used when starting the bot for the first time. You will receive a welcome message and usage instructions.
  </Card>
  <Card title="/pair code" icon="link">
    Used to connect your ClawOnPhone account with Telegram. Enter the pairing code generated in ClawOnPhone along with this command.
  </Card>
  <Card title="/status" icon="circle-check">
    Checks the current connection status. Use this to verify if your account is properly connected.
  </Card>
  <Card title="/help" icon="circle-question">
    Shows the list of available commands and help information.
  </Card>
  <Card title="/clear" icon="rotate">
    Clears the conversation history. Veta will forget the previous conversation and start a new one.
  </Card>
</CardGroup>

<Note>
  To use Google-related tools (Email, Calendar, Files, etc.) in Telegram, you must first connect your Google account in the **ClawOnPhone app**. Open ClawOnPhone and press the **G button** on the screen to connect it.
</Note>

<Tip>
  You only need to set up the Telegram connection once. Afterwards, you can simply open Telegram and send messages directly to the ClawOnPhone Bot.
</Tip>
