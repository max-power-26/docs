## Before Installation

You can install ClawOnPhone directly from your internet browser. There is no need to visit the App Store or Play Store.

Please enter the URLs below in your browser.

- **ClawOnPhone** (AI Agent): https://alarm-e59f0.web.app
- **ClawTalk** (Business Messenger): https://url-saas-240311.web.app

<Tabs>
  <Tab title="Android">
    On Android phones, please use the **Chrome browser**.

    <Steps>
      <Step title="Enter Address in Chrome">
        Open the Chrome browser and enter the app URL from above into the address bar.
      </Step>
      <Step title="Check the Installation Prompt">
        When the page opens, an **"Add to Home screen"** notification banner will appear at the bottom of the screen. If you see this banner, tap it right away. If the banner doesn't appear, follow the next step.
      </Step>
      <Step title="Install via Menu">
        Tap the three-dot menu button in the top right corner of the screen. In the menu, tap **"Add to Home screen"** or **"Install app"**.
      </Step>
      <Step title="Installation Complete">
        When you tap the confirmation button, an app icon will be created on your home screen. Now you can launch it exactly like a regular app by tapping the icon.
      </Step>
    </Steps>

    <Tip>
      ClawOnPhone and ClawTalk are separate apps. You can install both using the same method.
    </Tip>
  </Tab>
  <Tab title="iPhone">
    On iPhones, you must use the **Safari browser**.

    <Warning>
      It will not install from other browsers such as Chrome, the Naver app, or in-app browsers like KakaoTalk. Please make sure to open it in Safari.
    </Warning>

    <Steps>
      <Step title="Enter Address in Safari">
        Open the Safari browser and enter the app URL from above into the address bar.
      </Step>
      <Step title="Tap the Share Button">
        Tap the **Share button** (a square with an arrow pointing up) located in the middle at the bottom of the screen.
      </Step>
      <Step title="Select Add to Home Screen">
        When the share menu appears, scroll down and find the **"Add to Home Screen"** option, then tap it.
      </Step>
      <Step title="Confirm Name and Add">
        Check the app name and tap the **"Add"** button in the top right corner.
      </Step>
      <Step title="Installation Complete">
        The app icon will be created on your home screen. You can tap the icon to launch it directly.
      </Step>
    </Steps>

    <Tip>
      If you received the link via messengers like KakaoTalk, long-press the link to copy the URL, then paste it directly into the Safari address bar.
    </Tip>
  </Tab>
  <Tab title="PC (Computer)">
    On taking PCs, please use the **Chrome browser**.

    <Steps>
      <Step title="Enter Address in Chrome">
        Open the Chrome browser and enter the app URL from above into the address bar.
      </Step>
      <Step title="Check the Install Icon">
        At the far right of the address bar, an **Install icon** (shaped like a monitor with a downward arrow) will appear.
      </Step>
      <Step title="Click the Install Button">
        When you click the Install icon, a confirmation window asking "Install app?" will appear. Please click the **"Install"** button.
      </Step>
      <Step title="Installation Complete">
        After the installation is finished, the app will open in a separate window. You can also find and run the app from your desktop or start menu.
      </Step>
    </Steps>

    <Tip>
      If the install icon is not visible, look for the "Install app" option in the Chrome menu (three dots).
    </Tip>
  </Tab>
</Tabs>

## Cannot Install?

If you are having trouble installing, please check the following.

<AccordionGroup>
  <Accordion title="The installation banner or icon does not appear">
    If your browser version is outdated, the installation feature might not be supported. Please update Chrome to the latest version on Android and PC. For iPhone, check for software updates in your iOS settings.
  </Accordion>
  <Accordion title="I already installed it, but the icon is missing">
    Try looking on other pages of your home screen or within the app drawer. On a PC, try searching for the app name in the Start menu.
  </Accordion>
  <Accordion title="I opened it with Chrome on iPhone but it won't install">
    On iPhone, adding to the home screen is only available strictly through Safari. Please copy the URL and open it directly in Safari.
  </Accordion>
  <Accordion title="The internet connection is unstable">
    Please verify that Wi-Fi or mobile data is properly connected. A stable internet connection is required during installation.
  </Accordion>
</AccordionGroup>

If the issue persists, please contact the administrator.
