## 音声でファイルやドキュメントを利用する

Vetaに話しかければ、Googleドライブからファイルを検索したり、Googleドキュメントで文書を作成したり、Googleスプレッドシートのデータを確認・修正したりできます。

<Note>
この機能を使用するには、**Googleアカウントの連携**が必要です。まだ連携していない場合は、まずGボタンを押してGoogleアカウントを連携してください。
</Note>

## Drive - ファイル検索

Googleドライブに保存されたファイルを名前やキーワードで検索できます。

**話しかけ方の例**

<CardGroup cols={2}>

<Card title="ファイル名で検索" icon="magnifying-glass">
**「見積書のファイルを探して」**

**「2月の報告書ファイルを検索して」**
</Card>

<Card title="種類で検索" icon="file">
**「最近作成したスプレッドシートを探して」**

**「テジュン製薬に関連するドキュメントはある？」**
</Card>

</CardGroup>

Vetaが検索結果をファイル名や更新日と一緒に教えてくれます。希望するファイルが見つかったら、「そのファイルを開いて」と言うだけで直接開くことができます。

## Docs - ドキュメントの読み取り・作成

Googleドキュメントの内容を確認したり、新しいドキュメントを作成したりできます。

**話しかけ方の例**

<CardGroup cols={2}>

<Card title="ドキュメントの読み取り" icon="file-lines">
**「企画書ドキュメントの内容を読んで」**

**「昨日作成した議事録を見せて」**
</Card>

<Card title="ドキュメントの作成" icon="file-pen">
**「議事録のドキュメントを作って」**

**「2月の営業報告書を作成して」**
</Card>

</CardGroup>

## 対話形式でのドキュメント作成

Vetaと対話しならが、ステップバイステップでドキュメントの内容を作成できます。

<Steps>

<Step title="ドキュメントの作成をリクエストします">
**私**: 「議事録のドキュメントを作って」

**Veta**: 「どのような内容を含めますか？ 会議の日付と参加者を教えてください。」
</Step>

<Step title="内容を伝えます">
**私**: 「今日の午前のチームミーティングで、参加者は私、キム部長、イ課長。主な議題は2月の売上確認だよ」

**Veta**: 「議事録ドキュメントを作成しました。タイトルは『2月23日 チームミーティング議事録』です。」
</Step>

</Steps>

## Sheets - スプレッドシートの読み書き

Googleスプレッドシートのデータを読み取ったり、新しいデータを追加したりできます。

**話しかけ方の例**

<CardGroup cols={2}>

<Card title="データの読み取り" icon="table">
**「売上シートを読んで」**

**「在庫状況のスプレッドシートを見せて」**
</Card>

<Card title="データの書き込み" icon="pen-to-square">
**「売上シートに今月の実績を追加して」**

**「在庫シートを更新して」**
</Card>

</CardGroup>

Vetaがシートのデータを読み取り、内容を整理して伝えてくれます。データを追加する際は、どのような値を入れるかを対話を通じて確認します。

## その他の話しかけ方の例

<CardGroup cols={1}>

<Card title="ファイルとドキュメントに関するコマンド" icon="folder-open">
**「ドライブからExcelのファイルを探して」**

**「先月の売上報告書のドキュメントを開いて」**

**「発注書のシートからA列のデータを読んで」**
</Card>

</CardGroup>

<Tip>
ファイル名を正確に覚えていなくても大丈夫です。「見積書」「報告書」のようなキーワードだけを言えば、Vetaが関連するファイルをすべて探し出してくれます。
</Tip>

<Tip>
Sheetsでデータを読むときに「A1からD10まで」のように範囲を指定すると、必要な部分だけを正確に確認できます。範囲を指定しない場合は、Vetaが全体の内容を要約して教えてくれます。
</Tip>
