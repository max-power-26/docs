## 설치 전 확인사항

ClawOnPhone은 인터넷 브라우저에서 바로 설치할 수 있습니다. 앱스토어나 플레이스토어에 방문할 필요가 없습니다.

아래 주소를 브라우저에 입력해 주세요.

- **ClawOnPhone** (AI 에이전트) : https://alarm-e59f0.web.app
- **ClawTalk** (업무 메신저) : https://url-saas-240311.web.app

<Tabs>
  <Tab title="안드로이드">
    안드로이드 폰에서는 **Chrome 브라우저**를 사용해 주세요.

    <Steps>
      <Step title="Chrome에서 주소 입력">
        Chrome 브라우저를 열고, 위의 앱 주소를 주소창에 입력합니다.
      </Step>
      <Step title="설치 안내 확인">
        페이지가 열리면 화면 아래쪽에 **"홈 화면에 추가"** 안내 배너가 나타납니다. 배너가 보이면 바로 눌러 주세요. 배너가 보이지 않으면 다음 단계를 따라 주세요.
      </Step>
      <Step title="메뉴에서 설치하기">
        화면 오른쪽 위의 점 세 개 메뉴 버튼을 눌러 주세요. 메뉴에서 **"홈 화면에 추가"** 또는 **"앱 설치"** 항목을 눌러 주세요.
      </Step>
      <Step title="설치 완료">
        확인 버튼을 누르면 홈 화면에 앱 아이콘이 생깁니다. 이제 일반 앱처럼 아이콘을 눌러서 실행할 수 있습니다.
      </Step>
    </Steps>

    <Tip>
      ClawOnPhone과 ClawTalk은 별도의 앱입니다. 두 가지 모두 같은 방법으로 설치하시면 됩니다.
    </Tip>
  </Tab>
  <Tab title="아이폰">
    아이폰에서는 반드시 **Safari 브라우저**를 사용해야 합니다.

    <Warning>
      Chrome, 네이버 앱, 카카오톡 인앱 브라우저 등 다른 브라우저에서는 설치가 되지 않습니다. 반드시 Safari에서 열어 주세요.
    </Warning>

    <Steps>
      <Step title="Safari에서 주소 입력">
        Safari 브라우저를 열고, 위의 앱 주소를 주소창에 입력합니다.
      </Step>
      <Step title="공유 버튼 누르기">
        화면 아래쪽 가운데에 있는 **공유 버튼**(네모에서 화살표가 위로 나오는 모양)을 눌러 주세요.
      </Step>
      <Step title="홈 화면에 추가 선택">
        공유 메뉴가 나타나면 아래로 스크롤하여 **"홈 화면에 추가"** 항목을 찾아 눌러 주세요.
      </Step>
      <Step title="이름 확인 후 추가">
        앱 이름을 확인하고 오른쪽 위의 **"추가"** 버튼을 눌러 주세요.
      </Step>
      <Step title="설치 완료">
        홈 화면에 앱 아이콘이 생깁니다. 아이콘을 눌러서 바로 실행할 수 있습니다.
      </Step>
    </Steps>

    <Tip>
      카카오톡 등에서 링크를 받았다면, 링크를 길게 눌러 주소를 복사한 뒤 Safari 주소창에 직접 붙여넣기 해 주세요.
    </Tip>
  </Tab>
  <Tab title="PC (컴퓨터)">
    PC에서는 **Chrome 브라우저**를 사용해 주세요.

    <Steps>
      <Step title="Chrome에서 주소 입력">
        Chrome 브라우저를 열고, 위의 앱 주소를 주소창에 입력합니다.
      </Step>
      <Step title="설치 아이콘 확인">
        주소창 오른쪽 끝에 **설치 아이콘**(모니터에 화살표 모양)이 나타납니다.
      </Step>
      <Step title="설치 버튼 클릭">
        설치 아이콘을 클릭하면 "앱을 설치하시겠습니까?" 확인 창이 나타납니다. **"설치"** 버튼을 클릭해 주세요.
      </Step>
      <Step title="설치 완료">
        설치가 끝나면 별도 창으로 앱이 열립니다. 바탕화면이나 시작 메뉴에서도 앱을 찾아 실행할 수 있습니다.
      </Step>
    </Steps>

    <Tip>
      설치 아이콘이 보이지 않으면, Chrome 메뉴(점 세 개)에서 "앱 설치" 항목을 찾아 주세요.
    </Tip>
  </Tab>
</Tabs>

## 설치가 안 되나요?

설치가 잘 되지 않는 경우 아래 사항을 확인해 주세요.

<AccordionGroup>
  <Accordion title="설치 배너나 아이콘이 나타나지 않아요">
    브라우저 버전이 오래된 경우 설치 기능이 지원되지 않을 수 있습니다. 안드로이드와 PC는 Chrome을 최신 버전으로 업데이트해 주세요. 아이폰은 iOS 설정에서 소프트웨어 업데이트를 확인해 주세요.
  </Accordion>
  <Accordion title="이미 설치했는데 아이콘이 안 보여요">
    홈 화면의 다른 페이지나 앱 서랍에서 찾아보세요. PC의 경우 시작 메뉴에서 앱 이름으로 검색해 보세요.
  </Accordion>
  <Accordion title="아이폰에서 Chrome으로 열었는데 설치가 안 돼요">
    아이폰에서는 반드시 Safari에서만 홈 화면 추가가 가능합니다. 주소를 복사한 뒤 Safari에서 직접 열어 주세요.
  </Accordion>
  <Accordion title="인터넷 연결이 불안정해요">
    Wi-Fi 또는 모바일 데이터가 정상적으로 연결되어 있는지 확인해 주세요. 설치 시 안정적인 인터넷 연결이 필요합니다.
  </Accordion>
</AccordionGroup>

그래도 문제가 해결되지 않으면 관리자에게 문의해 주세요.
