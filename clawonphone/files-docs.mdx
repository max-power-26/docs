## 음성으로 파일과 문서 사용하기

Veta에게 말하면 Google Drive에서 파일을 검색하고, Google Docs로 문서를 만들고, Google Sheets의 데이터를 확인하거나 수정할 수 있습니다.

<Note>
이 기능을 사용하려면 **Google 계정 연결**이 필요합니다. 아직 연결하지 않았다면 먼저 G 버튼을 눌러 Google 계정을 연결해 주세요.
</Note>

## Drive - 파일 검색

Google Drive에 저장된 파일을 이름이나 키워드로 검색할 수 있습니다.

**말하기 예시**

<CardGroup cols={2}>

<Card title="파일명으로 검색" icon="magnifying-glass">
**"견적서 파일 찾아줘"**

**"2월 보고서 파일 검색해줘"**
</Card>

<Card title="종류로 검색" icon="file">
**"최근에 만든 스프레드시트 찾아줘"**

**"태준제약 관련 문서 있어?"**
</Card>

</CardGroup>

Veta가 검색 결과를 파일명, 수정 날짜와 함께 알려줍니다. 원하는 파일을 찾으면 "그 파일 열어줘"라고 말해서 바로 열 수 있습니다.

## Docs - 문서 읽기 및 작성

Google Docs 문서의 내용을 확인하거나 새 문서를 만들 수 있습니다.

**말하기 예시**

<CardGroup cols={2}>

<Card title="문서 읽기" icon="file-lines">
**"기획서 문서 내용 읽어줘"**

**"어제 만든 회의록 보여줘"**
</Card>

<Card title="문서 작성" icon="file-pen">
**"회의록 문서 만들어줘"**

**"2월 영업 보고서 문서 작성해줘"**
</Card>

</CardGroup>

## 대화형 문서 작성

문서 내용을 Veta와 대화하면서 작성할 수 있습니다.

<Steps>

<Step title="문서 작성을 요청합니다">
**나**: "회의록 문서 만들어줘"

**Veta**: "어떤 내용을 담을까요? 회의 날짜와 참석자를 알려주세요."
</Step>

<Step title="내용을 알려줍니다">
**나**: "오늘 오전 팀 미팅이고, 참석자는 나, 김부장, 이과장이야. 주요 안건은 2월 매출 점검"

**Veta**: "회의록 문서를 만들었습니다. 제목은 '2월 23일 팀 미팅 회의록'입니다."
</Step>

</Steps>

## Sheets - 스프레드시트 읽기 및 쓰기

Google Sheets의 데이터를 읽거나, 새로운 데이터를 추가할 수 있습니다.

**말하기 예시**

<CardGroup cols={2}>

<Card title="데이터 읽기" icon="table">
**"매출 시트 읽어줘"**

**"재고 현황 스프레드시트 보여줘"**
</Card>

<Card title="데이터 쓰기" icon="pen-to-square">
**"매출 시트에 이번 달 실적 추가해줘"**

**"재고 시트 업데이트해줘"**
</Card>

</CardGroup>

Veta가 시트의 데이터를 읽어서 내용을 정리해 알려줍니다. 데이터를 추가할 때는 어떤 값을 넣을지 대화를 통해 확인합니다.

## 더 많은 말하기 예시

<CardGroup cols={1}>

<Card title="파일과 문서 관련 명령" icon="folder-open">
**"드라이브에서 엑셀 파일 찾아줘"**

**"지난달 매출 보고서 문서 열어줘"**

**"발주서 양식 시트에서 A열 데이터 읽어줘"**
</Card>

</CardGroup>

<Tip>
파일 이름을 정확히 기억하지 못해도 괜찮습니다. "견적서", "보고서"처럼 키워드만 말하면 Veta가 관련 파일을 모두 찾아줍니다.
</Tip>

<Tip>
Sheets에서 데이터를 읽을 때 "A1부터 D10까지"처럼 범위를 지정하면 원하는 부분만 정확하게 볼 수 있습니다. 범위를 지정하지 않으면 Veta가 전체 내용을 요약해서 알려줍니다.
</Tip>
