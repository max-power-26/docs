## 음성으로 웹 검색과 날씨 확인하기

Veta에게 말하면 인터넷에서 정보를 검색하거나, 날씨와 현재 시간을 바로 확인할 수 있습니다. 이 기능은 **Google 계정 연결 없이도** 바로 사용할 수 있습니다.

## 웹 검색

궁금한 내용을 물어보면 Veta가 인터넷에서 최신 정보를 찾아 알려줍니다.

**말하기 예시**

<CardGroup cols={2}>

<Card title="일반 검색" icon="magnifying-glass">
**"애플 주가 검색해줘"**

**"삼성전자 최신 뉴스 알려줘"**
</Card>

<Card title="업무 관련 검색" icon="briefcase">
**"의약품 유통 규정 검색해줘"**

**"GMP 인증 기준 알려줘"**
</Card>

</CardGroup>

Veta가 검색 결과를 요약해서 알려줍니다. 더 자세한 내용이 필요하면 "좀 더 자세히 알려줘"라고 이어서 말하면 됩니다.

## 날씨 확인

원하는 지역의 날씨를 바로 확인할 수 있습니다.

**말하기 예시**

<CardGroup cols={2}>

<Card title="현재 날씨" icon="sun">
**"서울 날씨 어때?"**

**"오늘 날씨 알려줘"**
</Card>

<Card title="다른 지역 날씨" icon="cloud">
**"부산 날씨 어때?"**

**"제주도 내일 날씨 알려줘"**
</Card>

</CardGroup>

현재 기온, 날씨 상태, 체감 온도 등을 알려줍니다.

## 현재 시간 확인

현재 시간을 음성으로 확인할 수 있습니다.

**말하기 예시**

<CardGroup cols={1}>

<Card title="시간 확인" icon="clock">
**"지금 몇 시야?"**
</Card>

</CardGroup>

## 활용 예시

웹 검색은 업무 중 빠르게 정보를 확인하고 싶을 때 유용합니다.

<CardGroup cols={2}>

<Card title="거래처 정보 확인" icon="building">
회의 전에 거래처 정보를 빠르게 검색할 수 있습니다.

**"태준제약 회사 정보 검색해줘"**
</Card>

<Card title="외근 전 날씨 확인" icon="umbrella">
외근 출발 전에 날씨를 확인하고 우산을 챙길 수 있습니다.

**"오늘 비 와?"**
</Card>

</CardGroup>

<Tip>
웹 검색 결과가 너무 많으면 질문을 구체적으로 해 보세요. "의약품"보다 "의약품 도매업 허가 기준 2026년"처럼 구체적으로 물어보면 더 정확한 결과를 받을 수 있습니다.
</Tip>

<Tip>
날씨, 시간 확인은 Google 계정 연결 없이도 바로 사용할 수 있습니다. 앱 설치 직후부터 바로 시도해 보세요.
</Tip>
